Active Directory

2025-2026

Description du Projet

Dans le cadre de ma formation en BTS SIO option SISR, j’ai mis en place une infrastructure Active Directory sous Windows Server. Le projet consistait à installer et configurer un contrôleur de domaine, structurer l’annuaire avec des Unités d’Organisation (OU), intégrer des postes clients et un serveur au domaine, puis gérer les utilisateurs et les groupes afin d’assurer une administration centralisée et sécurisée du réseau.

Objectifs Réalisés

  • Création de l'AD et de ses UO
  • Intégration de postes clients au domaine Active Directory
  • Jonction de Serveur au domaine Active Directory
  • Gestion des utilisateurs et des groupes

Technologies Utilisées

PowerShell Active Directory Windows Server Ubuntu Server

Résultats et Compétences Acquises

Ce projet m’a permis de maîtriser les bases de l’administration d’un domaine Active Directory : création et organisation des OU, gestion des comptes utilisateurs et des groupes de sécurité, intégration de postes Windows au domaine et configuration des services associés. J’ai également développé mes compétences en scripting PowerShell et en gestion d’un environnement Windows Server, tout en appliquant les bonnes pratiques d’administration et de sécurité.
