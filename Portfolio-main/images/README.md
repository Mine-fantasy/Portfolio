# Images des Projets

Ce dossier contient les images des projets affichées dans la section "Mes Projets" du portfolio.

## Structure

Les fichiers d'images doivent être nommés comme suit :

- **projet-infra.jpg** - Image du projet "Infrastructure Virtualisée"
- **projet-securite.jpg** - Image du projet "Sécurisation Réseau"
- **projet-automatisation.jpg** - Image du projet "Automatisation Scripts"

## Recommandations

- **Format** : JPG ou PNG (JPG recommandé pour les photos/captures d'écran)
- **Dimensions** : Minimum 400x250 pixels (aspect ratio 16:9 recommandé)
- **Taille du fichier** : Max 500 KB pour optimiser le chargement
- **Contenu** : 
  - Une capture d'écran du projet
  - Un diagramme de l'infrastructure
  - Une visualisation du code
  - Un logo ou une image représentative

## Comment ajouter une image

1. Prépare une image au format JPG ou PNG
2. Redimensionne-la à environ 800x500 pixels (ou garde l'aspect ratio 16:9)
3. Nomme le fichier correctement (exemple: `projet-infra.jpg`)
4. Place le fichier dans ce dossier (`images/`)
5. Les images s'afficheront automatiquement dans le portfolio

## Exemple

Si tu veux remplacer l'image du projet "Infrastructure Virtualisée" :
1. Crée une capture d'écran de ton projet
2. Export en JPG
3. Renomme en `projet-infra.jpg`
4. Place dans le dossier `images/`
5. Rafraîchis la page du portfolio

Les images s'ajusteront automatiquement à la taille de la carte (200px de hauteur).
