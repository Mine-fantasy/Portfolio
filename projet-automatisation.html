Automatisation Scripts

2024 - 2025

Description du Projet

Développement d'une suite de scripts pour automatiser les tâches administratives IT répétitives. Ce projet visait à améliorer l'efficacité opérationnelle en réduisant le travail manuel et les risques d'erreurs.

Objectifs Réalisés

  • Script PowerShell pour automatiser les sauvegardes de la configuration des Switch
  • Solution utilisé pour l'envoie des configurations vers tftp: Archive
  • Script PowerShell pour trier les fichiers de config par date d'arriver supprimer les fichiers les plus anciens
  • tâches planifiées pour éxécuter le script de tri

Technologies Utilisées

PowerShell Bash IOS cisco Archive Tache planifié

Résultats et Compétences Acquises

Grâce à ce projet, j'ai maîtrisé la programmation de scripts pour l'automatisation de taches sur un switch Cisco. J'ai consolidé mes compétences en PowerShell, et développé une bonne compréhension et connaissance de l'IOS cisco.
